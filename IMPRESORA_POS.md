## 🔧 Cómo Usar

### Paso 1: Conecta la Impresora
```
1. Conecta por USB a tu computadora
2. Instala los drivers (generalmente se instalan automáticamente)
3. Prueba una impresión de prueba desde Windows
```

### Paso 2: Configura en Windows (Opcional)
```
1. Panel de Control → Dispositivos e Impresoras
2. Busca tu impresora en la lista
3. Establécela como impresora predeterminada (opcional)
```

### Paso 3: Usa la Aplicación
```
1. Abre CocinaFlow
2. Cambia un pedido a "Entregado"
3. ¡Se abrirá automáticamente el diálogo de impresión!
4. Selecciona tu impresora POS
5. Click en "Imprimir"
```
